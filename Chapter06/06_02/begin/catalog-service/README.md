# Catalog Service

## Application tasks

**Build the application**

```bash
./gradlew build
```

**Run the application (JAR)**

```bash
java -jar build/libs/catalog-service-0.0.1-SNAPSHOT.jar
```

**Run the application (Boot)**

```bash
./gradlew bootRun
```

**Run tests**

```bash
./gradlew test
```
